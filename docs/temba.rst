temba package
=============

To create a client instance, you need to know the name of the host server, and your API token. If you don't know your API token then visit the `API Explorer <http://rapidpro.io/api/v1/explorer>`_. For example:

.. code-block:: python

    from temba import TembaClient
    client = TembaClient('rapidpro.io', <YOUR-API-TOKEN>)
